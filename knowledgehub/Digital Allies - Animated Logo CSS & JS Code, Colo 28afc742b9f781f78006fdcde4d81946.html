How the Logo Works

Your logo consists of three components:

  1. Animated dot - A blue circular dot that pulses with a breathing animation
  1. Text - "Digital Allies" text with a gradient effect from blue to green
  1. Animation - Smooth pulse effect that cycles every 3 seconds

Color Specifications

  • Primary Blue (dot): #438eff
  • Gradient Start: #2563EB (blue)
  • Gradient End: #059669 (green)

Animation Details

The dot uses the brand-pulse animation which:

  • Scales from 1.0 to 1.2 (20% larger)
  • Fades from opacity 1.0 to 0.6
  • Completes one cycle every 3 seconds
  • Runs infinitely

Usage Notes

  • The logo works with Tailwind CSS utility classes
  • Use flex items-center space-x-1 for proper alignment
  • The dot is 8px (w-2 h-2) and animated
  • Text uses gradient effect with webkit-background-clip for browser compatibility
  • Footer version uses white color scheme for contrast
