CMS Architecture

Static Site Generation

  • Source format: Markdown files with YAML front matter
  • Output format: Static HTML pages
  • Template system: Shared layouts and components
  • Automation: GitHub Actions for content synchronization

Blog Post Creation Workflow

  1. Generate post scaffold with npm script
  1. Write content in Markdown with front matter
  1. Run sync command to generate HTML
  1. Commit changes for deployment

File Structure

content/
├── *.md                    # Source blog posts
├── blog-index.json         # Generated post index
└── posts/
    └── [slug]/
        └── index.html      # Generated post pages

Key Commands

Post Generation

npm run generate-post -- --title="Your Post Title" --description="Summary" --category="Web Development" --tags="html,css"

Content Synchronization

npm run sync-cms

Template Features

  • Shared header navigation
  • Post metadata (date, reading time, tags)
  • Markdown content rendering
  • Related posts suggestions
  • SEO optimization with meta tags
