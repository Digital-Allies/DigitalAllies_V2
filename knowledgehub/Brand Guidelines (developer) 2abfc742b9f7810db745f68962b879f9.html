Brand Guidelines (developer)

Brand Overview

Digital Allies is dedicated to empowering businesses and communities through accessible, high-quality digital solutions. We believe in making technology work for everyone.

Brand Values

  • Approachable: We make complex technology simple and accessible
  • Empowering: We give businesses the tools to succeed online
  • Community-Focused: We build lasting relationships and support local communities

Color Palette

Primary Colors

  • Primary Blue: #2563EB - Primary brand color, CTAs, links
  • Success Green: #059669 - Success messages, positive actions
  • Alert Red: #EF4444 - Important alerts, error states

Secondary Colors

  • Light Blue: #DBEAFE - Background accents, hover states
  • Light Green: #D1FAE5 - Success backgrounds
  • Dark Gray: #414243 - Secondary text, borders

Gradients

  • Primary: linear-gradient(135deg, #2563EB, #059669)
  • Blue: linear-gradient(135deg, #438eff, #2563eb)

Typography

Font Families

  • Primary: Segoe UI (Headlines, body text)
  • Secondary: Source Sans Pro (Subsections, small text)

Typography Scale

  • H1 Hero: 48px, Black (900)
  • H2 Section: 36px, Semibold (600)
  • H3 Subsection: 24px, Regular (400)
  • Body: 18px, Regular (400)
  • Small: 14px, Regular (400)

Logo Usage

Specifications

  • Minimum size: 100px width (horizontal), 48px height (stacked)
  • Clear space: Equal to dot diameter around logo
  • File formats: SVG (preferred), PNG with transparency
  • Backgrounds: Works on white, light colors, or transparent

Variants

  • Default: Blue-to-green gradient with animated pulsing dot
  • White: For dark backgrounds
  • Black: For light backgrounds
  • Offline: Red dot for offline/error states

Button Styles

Primary Buttons

  • Background: #2563EB
  • Text: #FFFFFF
  • Hover: #1d4ed8
  • Use for: Primary actions, CTAs

Secondary Buttons

  • Border: #2563EB (2px)
  • Text: #2563EB
  • Hover: #DBEAFE background
  • Use for: Secondary actions

Ghost Buttons

  • Text: #2563EB
  • Hover: #DBEAFE background
  • Use for: Tertiary actions, navigation

Icon Guidelines

Usage

  • Minimum size: 24px for UI, 48px for features
  • Colors: Blue (#2563EB), Green (#059669), Red (#EF4444)
  • Spacing: Minimum 8px padding around icons
  • Style consistency: Use same style throughout design

Available Icons

  • Web & Global
  • Design & Art
  • Consulting
  • Cloud Services
  • Community
  • Grid Organization
  • Lightbulb (Ideas)
  • SEO (Search)
  • Communication
  • Blog (Content)

Accessibility

  • Maintain WCAG AA standards
  • Minimum contrast: 4.5:1 for body text, 3:1 for large text
  • Touch targets: Minimum 44x44px
  • Clear focus indicators on interactive elements
  • Alt text for all images
